# GitHub Repository Plan

## Recommendation

Create a GitHub repository for this project.

Recommended repository:

```txt
promisenxu/promisexu.com
```

Use a public repository unless there is private content or unpublished writing that should not be visible before launch.

## Why A Repo Is Worth It

- Cloudflare can deploy automatically from GitHub.
- Content edits are versioned.
- Issues can track phases and bugs.
- Pull requests provide a review path.
- The repo itself is a proof point for engineering hygiene.

## Suggested Initial Repo Settings

- Visibility: public, unless drafts/private content are committed.
- Default branch: `main`.
- Branch protection: add after the first stable launch.
- Issues: enabled.
- Discussions: disabled unless needed later.
- Wiki: disabled.
